# My Process World

把 Claude Code 级别的自主智能体装进你的桌面。一个安装包，即装即用。数据不出本机、密钥自己掌握、模型随便换。它不只是聊天框，而是一个会自己干活的工程师。

## 下载安装

前往 [Releases 页面](../../releases/latest) 下载最新安装包：

MyProcessWorld-0.1.0.exe（Windows x64，约 76 MB，离线即装）

双击安装即可使用。首次使用进入「设置 - 大模型」，填入任意 OpenAI 兼容服务的 API Key 与模型名（DeepSeek、硅基流动、Moonshot、OpenAI 等），完成。

## 它能做什么

把一句话丢给它，例如"分析工作区里的数据文件，画出趋势图并总结规律"，然后看着它自己完成：规划待办清单、联网检索、读懂你的项目、写代码、运行、看报错、修复、再运行、交付结果。全程可视化，每一步都看得见。

## 核心能力

| 能力 | 说明 |
|---|---|
| 全自主智能体循环 | 规划、执行、验证、纠偏闭环；写完代码必须真跑通才算完 |
| 多智能体协作编排 | 总指挥按波次派出最多 6 个角色化子智能体（调研、采集、编码、数据、仿真、审查）并行分工，委派带明确目标、边界与验收标准，汇总后过质量门再交付 |
| 多引擎联网检索 | Google、Bing、百度、DuckDuckGo、GitHub 并行检索，按相关性合并去重，代理感知，内网环境也能查资料 |
| 运行中插话引导 | 任务运行时随时插话纠偏，智能体下一步立即采纳，绝不丢话 |
| MCP 生态 | 标准 Model Context Protocol 客户端，接入任意 MCP Server 扩展工具 |
| 项目级指令 | 自动读取 AGENTS.md、CLAUDE.md、copilot-instructions.md，进了你的项目就懂你的规矩 |
| 智能上下文压缩 | 超长任务自动摘要压缩历史，几十步、上百次工具调用不断片 |
| 长期记忆 | 用户偏好、环境事实、经验教训自动沉淀，踩过的坑下次自动规避 |
| 后台进程管理 | dev server、长训练任务丢后台跑，智能体边等边干别的 |
| 容错精确编辑 | 精确局部替换改大文件不翻车，每个文件可逐一审阅、保留或撤销 |
| 多会话并行标签 | 多个任务同时跑，标签页互不干扰，可关闭可续聊，历史不丢 |
| Python 与 Notebook | 逐 cell 执行 ipynb，画图直出对话流，内置 STL 三维查看器 |
| 本地隐私 | 仅监听 127.0.0.1，源码、数据、密钥不出本机 |
| 自带密钥 | OpenAI 协议兼容，任意模型想换就换 |

## 插话引导

智能体运行中，输入框自动变成插话通道，随手补充要求、纠偏、指方向，不打断运行。内置引导栏一键发送高频指令：汇报进展、加快收尾、换个思路、聚焦核心、少改东西、运行验证。插话拥有最高优先级，智能体立即采纳并确认响应方式；任务未就绪时自动排队，就绪即转达，一个字都不会丢。

## 设计哲学

1. 本地优先。你的代码和数据是你的，永远。
2. 真验证。智能体改完必须跑通，不交"看起来对"的活。
3. 人在回路。随时插话、随时引导、随时叫停，AI 干活你掌舵。
4. 越用越强。经验沉淀为长期记忆与技能，平台跟着你成长。
