## 🗣 插话引导：人机协作的正确姿势

别的工具发出任务就只能干等；这里你是**驾驶员**：

- 智能体运行中，输入框自动变成「插话」通道，随手补充要求、纠偏、指方向，不打断运行；
- 内置引导栏一键发送高频指令：📋 汇报进展 / ⏩ 加快收尾 / 🔄 换个思路 / 🎯 聚焦核心 / 🛡️ 少改东西 / 🧪 运行验证；
- 插话被赋予**最高优先级**：智能体立即采纳、更新待办、并确认它将如何响应；
- 任务还没就绪？插话自动排队，就绪即转达；模型正在收尾？插话照样被消费，**一个字都不会丢**。

## 💡 设计哲学

1. **本地优先**——你的代码和数据是你的，永远。
2. **真验证**——智能体改完必须跑通，不交“看起来对”的活。
3. **人在回路**——随时插话、随时引导、随时叫停，AI 干活你掌舵。
4. **越用越强**——经验沉淀成 Skill，记忆跨会话延续，平台跟着你成长。
