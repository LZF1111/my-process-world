# My Process World

把 Claude Code 级别的自主智能体装进你的桌面。一个安装包，即装即用。数据不出本机、密钥自己掌握、模型随便换。它不只是聊天框，而是一个会自己干活的工程师。

## 下载安装

前往 [Releases 页面](../../releases/latest) 下载最新安装包：

MyProcessWorld-0.3.0.exe（Windows x64，约 127 MB，离线即装，内置 MDriver 分子模拟智能体封装版）

双击安装即可使用。首次使用进入「设置 - 大模型」，填入任意 OpenAI 兼容服务的 API Key 与模型名（DeepSeek、硅基流动、Moonshot、OpenAI 等），完成。

## 它能做什么

把一句话丢给它，例如"分析工作区里的数据文件，画出趋势图并总结规律"，然后看着它自己完成：规划待办清单、联网检索、读懂你的项目、写代码、运行、看报错、修复、再运行、交付结果。全程可视化，每一步都看得见。

## 智能体大军与内置 MDriver

点左侧「⚔ 智能体大军」进入像素办公室：每个队员是一只小猪，各自独立模型与职责，忙时在工位敲代码，闲时去休息区睡觉、吃面、打赛车游戏。

安装包自带 **MDriver 封装版**（LAMMPS 分子模拟智能体，含官方案例库），首次启动自动绑定给「MD 算师」队员：

- 工具坞点「🖥 打开」：自动探活→拉起→打开界面，并把你在工作台配的 API Key/模型自动填进 MDriver（不覆盖已有配置），开箱即用。
- 对 MD 算师下令（如「跑个聚乙烯热解」）：它会自动连上 MDriver，像真人一样在它的输入框里打字、等结果、回报。
- 不需要时，工具坞点「✕」可解绑，并可选择顺带删除捆绑文件释放磁盘空间（只允许删安装包自带的示例，绝不动你自己的目录）。

## MCP 工具：新闻爬虫示例

「设置 → MCP 外部工具服务器」或智能体大军的工具坞，都有「🗞 一键接入新闻爬虫」：安装一个纯本地 MCP 脚本（`~/.processnet/mcp-examples/news-agent.mjs`，零依赖），暴露 `latest_news` 工具。装好后：

- AI 助手智能体模式里说「查一下今天的 AI 新闻」，模型会自动调用 `mcp__news__latest_news`。
- 智能体大军的总指挥与队员同样能调。
- 工具坞可「▶ 试用」手动查，也可「✕」随时删除。
- 脚本本身就是二次开发模板：照着它再写一个工具（天气、股价、内部系统…），在设置里加一行 JSON 即可接入。

## 插话引导

智能体运行中，输入框自动变成插话通道，随手补充要求、纠偏、指方向，不打断运行。内置引导栏一键发送高频指令：汇报进展、加快收尾、换个思路、聚焦核心、少改东西、运行验证。插话拥有最高优先级，智能体立即采纳并确认响应方式；任务未就绪时自动排队，就绪即转达，一个字都不会丢。

## 设计哲学

1. 本地优先。你的代码和数据是你的，永远。
2. 真验证。智能体改完必须跑通，不交"看起来对"的活。
3. 人在回路。随时插话、随时引导、随时叫停，AI 干活你掌舵。
4. 越用越强。经验沉淀为长期记忆与技能，平台跟着你成长。
